# 🌍 Interactive Web GIS – Netherlands Cities & Regions

🔗 **Live Demo:** https://gis.hcoco1.com

---

## 📌 Overview

This project is an **interactive Web GIS application** built using **Leaflet.js**, designed to visualize and explore major cities, corridors, and regions in the Netherlands.

It demonstrates core GIS concepts such as **data-driven styling, spatial filtering, clustering, and interactive querying**, combined with modern web development practices.

---

## 🚀 Features

* 🗺️ **Interactive Map (Leaflet)**
* 📍 **City Markers with Dynamic Scaling (by rank)**
* 🧩 **Marker Clustering for performance**
* 🔎 **Search Function (city lookup + zoom)**
* 🎚️ **Population Filter (real-time)**
* 🎯 **Auto Zoom to Filtered Results**
* 🧱 **Layer Control (Cities / Corridors / Regions)**
* 🎨 **Data-Driven Styling (GeoJSON-based)**
* 💬 **Interactive Popups**
* ✨ **Hover Highlight Effects**
* 🌐 **Custom Domain Deployment**

## ⚙️ How It Works

1. GeoJSON data is loaded dynamically using `fetch()`
2. Features are styled based on their properties
3. Cities are rendered as markers with custom icons
4. Clustering groups markers for performance
5. UI controls allow filtering, searching, and toggling layers
6. Map updates dynamically based on user interaction
